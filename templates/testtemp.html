About

Hello, and welcome to the Simple Search for Reddit client that we have created! This web application allows a user to choose a subreddit, amount, and type of data they want to view in a simple streamlined experience. With this project that our team has worked so hard on we have developed a tool for new redditors to use and try out. It allows you to view the posts and also view the media associated with it. On behalf of our team we hope you enjoy our project and use it.
